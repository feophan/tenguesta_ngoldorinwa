<Num index={chapter}/> The native writing systems for Quenya were devised by elves in Aman. The **script of Rúmil** went out of use before the classical period. On the other hand, the **script of Feanor** was brought by Noldor after the Exile with them to the Middle-earth, and there it was applied to a variety of languages.

<Num index={chapter}/> Other scripts include the runic script of **Angerthas Daeron** used by the elves in Eregion, and Latin, first used to transcribe elvish languages by the Anglo-Saxon traveler *Ælfwine*. With that, Latin now is the most common script to write in Quenya.

## Tengwar Script

<Num index={chapter}/> We will discuss three main systems of *tengwar* as used in Quenya:

+ as an **abugida**: each symbol is a consonant or a combination of consonants, followed by /a/;
+ as an **alphasyllabary**: each symbol is a consonant or a combination of consonants;
+ and as an **alphabet**: both consonants and vowels are drawn by independent symbols.

#### Tengwar as Abugida

<Num index={chapter}/> In the most common use each letter represents a consonant (or a cluster of them) followed by a sound /a/. Other vowels are marked as diacritics.

##### Consonants

<Num index={chapter}/> The four series denote the place of articulation: I dentals, II labials, III velars, IV labialized velars.

The grades in Parmaquesta period denoted the manner of articulation: 1 voiceless stops, 2 nasalized voiced stops, 3 voiceless fricatives, 4 nasalized voiceless stops, 5 nasals, 6 approximants.

In Tarquesta some symbols changed their values or went out of use, so this system doesn't strictly hold true.

##### Recommendations for Print

<Num index={chapter}/> There is a great variation in orthography, as tengwar was never standartized. Here are a few guidelines for its use in a print medium.

<Num index={chapter}/> Use of letters with the same values:

+ <span class="tengwar"></span> is still used to denote old voiceless liquids which disappeared in Noldorin. <span class="tengwar"></span> is used initially and <span class="tengwar"></span> medially: <span class="tengwar"></span> /raj/, <span class="tengwar"></span> /hanːo/, <span class="tengwar"></span> /aha/.
+ <span class="tengwar"></span> is used before a consonant, and <span class="tengwar"></span> before a vowel: <span class="tengwar"></span> /raːma/, <span class="tengwar"></span> /kerta/, <span class="tengwar"></span> /orto/.
+ <span class="tengwar"></span> is used before a consonant, and <span class="tengwar"></span> before a vowel: <span class="tengwar"></span> /javanːa/, <span class="tengwar">/</span> /ujnen/, <span class="tengwar"></span> /ajːa/.
+ <span class="tengwar"></span> is used before a consonant, and <span class="tengwar"></span> after: <span class="tengwar"></span> /awre/, <span class="tengwar"></span> /vilwa/, <span class="tengwar"></span> /nawːe/.
+ <span class="tengwar"></span> is only used with *yá-tehta* as /j/.
+ Etymological spelling of initial <span class="tengwar"></span> is **not** maintained.
+ Etymological spelling of <span class="tengwar"></span> **is** maintained.

:::note[Late TQ]
Eventually the etymological spelling was abandoned, and <span class="tengwar"></span>, <span class="tengwar"></span> became used for /st/, /χt/, while the long-stemmed <span class="tengwar"></span>, <span class="tengwar"></span>, <span class="tengwar"></span> were discarded.
:::

<Num index={chapter}/> Use of marks:

1. The swashes (or *rince*) for the final /n/, /s/, and /r/ are **not** used.
2. The *anda-tehta* and *yá-tehta* should be placed **below** the letter.
4. To avoid mark-stacking issues, *wá-tehta* is **not** used.

#### Tengwar as Alphasyllabary

<Num index={chapter}/> The only difference with the previous mode is that /a/ is not assumed after every letter. Instead, a special diacritic <span class="tengwar"></span> is employed, while <span class="tengwar"></span> has no use. This mode was used in ornamental and decorated writing.

##### Mixed system

An impure alphabetical system that, like abugida, assumed a following /a/ after a consonant, was particularly common in Rivendell. In that system <span class="tengwar"></span> was used for <span class="tengwar"></span> as in abugida mode, and for /o/ <span class="tengwar"></span> was used instead.

:::caution
In the *Morphology* part, the true alphabetic system is used to clearly present the inflectional endings. However, in the *Syntax* part, the more common mixed system is employed for example sentences.
:::

### Traditional Transcription

<Num index={chapter}/> This method is mostly identical with phonetical transcription, but borrows a few conventions from classical spelling (TBD):

+ ⟪k⟫ is spelled ⟨c⟩;
+ ⟪kw⟫ is spelled ⟨qu⟩;
+ ⟪þ⟫ is spelled as ⟨th⟩ if (rarely) distinguished from ⟨s⟩.
+ nasal vowels are not spelled out;
+ final /e/ is spelled ⟨ë⟩;
+ the first vowel in hiatus is marked with diaresis: ⟨ëa⟩, ⟨öa⟩, etc. If the first letter is capital, the second one takes diaresis: ⟨Eä⟩, ⟨Oä⟩, etc.

#### Modified Traditional Transcription
